Error codes

The server and in some cases API clients will return these codes when requests do not work properly.

Code Section Description
0 All No problem
1 All A general error occurred, this often means a communication problem
2 All Invalid game credentials. Make sure you set your public and private keys up properly in your database
3 All Request timed out
4 All Invalid request
100 GeoIP GeoIP API has been disabled for this game
200 Leaderboards Leaderboard API has been disabled for this game
201 Leaderboards The player's name was not provided when saving a score
203 Leaderboards Player is banned from submitting scores
204 Leaderboards Score was not saved because it was not the player's best, you can allow players to have more than one score by specifying allowduplicates=true in your save options
300 GameVars GameVars API has been disabled for this game
400 PlayerLevels PlayerLevel API has been disabled for this game
401 PlayerLevels Invalid rating value (must be 1 - 10)
402 PlayerLevels Player has already rated this level
403 PlayerLevels Missing level name
404 PlayerLevels Missing level id
405 PlayerLevels Level already exists
500 Achievements Achievements API has been disabled for this game.
501 Achievements Missing playerid
502 Achievements Missing player name
503 Achievements Missing achievement
504 Achievements Invalid achievement or achievement key
505 Achievements Player already had the achievement. You can overwrite old achievements with overwrite=true or save each time the player is awarded with allowduplicates=true
506 Achievements Player already had the achievement and it was overwritten or a duplicate was saved successfully
600 Newsletter Newsletter API has been disabled for this game.
601 Achievements MailChimp API key is not configured
602 Newsletter MailChimp API returned an error
